# 项目介绍

本项目使用【Ant Design Pro】进行初始化。

## 环境准备

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

## 提供的脚本

Ant Design Pro 提供了一些有用的脚本来帮助您快速启动和构建 Web 项目、代码样式检查和测试。 `package.json` 中提供的脚本。修改或添加其他脚本是安全的：

### 开始项目

```bash
npm start
```

### 构建打包

```bash
npm run build
```

### 检查

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### 测试

```bash
npm test
```
